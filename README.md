# Multi-Product Recognition & Price Estimation (DL Project)

## Overview
This project implements image classification using multiple CNN architectures
to recognize fruits and vegetables and estimate total checkout price.

## Models Trained
- ResNet50
- VGG16
- AlexNet
- InceptionV3
- MobileNetV3

## Metrics Compared
- Top-1 Accuracy
- Top-5 Accuracy
- Macro F1
- Inference Speed (FPS)
- Model Size

## Web Demo
Upload an image → model predicts product → system calculates total price.

## Tech Stack
- PyTorch
- FastAPI
- Jinja2
